# Tentang Project

Website portfolio fotografi personal milik **Muhammad Fajar**, seorang pelajar yang memiliki ketertarikan pada dunia fotografi dan dokumentasi visual.

Portfolio ini menampilkan hasil foto pilihan yang diambil menggunakan:

- Canon SX410 IS
- Redmi 13

Website dibuat menggunakan teknologi web dasar tanpa framework sehingga ringan, cepat, dan mudah dipelajari.
